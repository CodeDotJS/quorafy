## Usage

```
 Usage   : quorafy [command] <username>

 Command :
  -u, --user      Get general information about Quora user
  -d, --download  Download profile picture of a Quora user

 Example :
 $ quorafy -u Adam-DAngelo
 $ quorafy -d Charlie-Cheever
```
